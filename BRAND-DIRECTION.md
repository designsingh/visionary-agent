# PageGrab Brand Direction

**Best brand territory:** *"Internet utility from an alternate 1998"* — polished for now.

---

## Principles

- **Simple UI-first branding** — let the interface do the talking
- **Strong iconography** — clear, purposeful icons
- **Personality from interface details** — witty micro-copy, thoughtful states
- **Nostalgia through structure, not through mess** — ordered, not chaotic

---

## Visual Direction

### 1. Windowed UI

Use interface frames as part of the brand:

- Floating app windows
- Title bars
- Old dialog box structure
- Segmented panels
- Sharp borders
- Inset buttons

*But cleaner and airier than actual 90s UI.*

### 2. Selection / Capture Language

Since the product is "grabbing" pages, this is where retro styling becomes useful:

- Dashed selection rectangles
- Crosshair cursor
- Crop marks
- Highlighted areas
- Drag-to-select frames
- "Captured" stamps

*Ties directly to product behavior — better than random retro styling.*

### 3. Pixel Restraint

Use pixel cues lightly:

- Tiny pixel icons
- Bitmap-ish accents
- Mono/pixel numerals in metadata
- 1px borders

*Do not make the whole thing chunky pixel art — avoid novelty-first.*

### 4. Old Web Colors, Modernized

**Route A: Terminal Utility** (recommended)
- Warm grey
- Charcoal
- Soft blue
- Acid green accent

**Route B: Browser Classic**
- Off-white
- Navy
- Medium grey
- Hyperlink blue

**Route C: Old Mac-ish**
- Light grey
- Black
- Platinum
- One playful accent (orange or teal)

---

## Brand Personality

PageGrab should feel:

- Clever
- Efficient
- Lightly nerdy
- Internet-native
- Not trying too hard
- A little witty

**Not:** Whimsical. Artisanal. Overloaded.
