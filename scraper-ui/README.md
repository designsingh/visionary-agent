# PageGrab

Capture any webpage as screenshots, markdown, and HTML. No sign-up, no API keys.

## Development

```sh
cd scraper-ui
npm install
npm run dev
```

Frontend: http://localhost:8080  
Backend API: http://localhost:3333 (run `npm run ui` from project root)

## Build

```sh
npm run build
```

Outputs to `../public` for deployment.

## Tech

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
